Пузырьковая сортировка

Данные отсортированы!

Показать инструкцию

  1. Нажмите на кнопку "Новые данные" для формирования набора из 10 случайных чисел с минимумом 1 и максимумом 50. Появится кнопка "Сортировать".

  2. При нажатии "Сортировать" начнётся процесс сортировки чисел по возрастанию. В синем круге будут подсвечены пары чисел, которые сравниваются в данный момент.

    Процесс может занять несколько минут. После его окончания отсортированный набор появится в отдельном поле для удобства копирования.

Прервать сортировку можно нажав "Новые данные". Старые данные при этом будут заменены и утеряны, а сортировку можно будет начать заново.
